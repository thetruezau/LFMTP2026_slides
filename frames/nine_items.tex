\begin{frame}
  \frametitle{Formalization using Autosubst 1}

  Why did we use the Autosubst 1?
  \begin{itemize}
  \item<2-> A library!
  \item<3-> Simple to learn and use (based on the $\sigma$-calculus).
  \item<4-> Minimalistic architecture that allows some modularity and customizability.
  \end{itemize}
  
  What is missing (for us)?
  \begin{itemize}
  \item<5-> More advanced tactics to derive substituion operations (how?).
  \item<6-> Modularization on the substitution lemmas.
  \item<7-> Support for interactions between multiple systems and subsystems.
  \item<8-> Of course support for mutually dependent syntactic classes would be good...
  \end{itemize}
  
\end{frame}
